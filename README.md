
# Project in Introduction to Computer Science 2026
This project is a Java-based desktop application using Java Swing to implement the Connect 4 game. 


## Description
Connect 4 is a two-player game where players take turns dropping colored discs into a grid.  
The discs fall to the lowest empty spot in the selected column.  
The goal is to connect four discs in a row, either horizontally, vertically, or diagonally.  
If the board becomes full before a player connects four discs, the game ends in a draw.


## Installation
1. Clone the repository
2. Open the project in your Java IDE
3. Compile and run the application
